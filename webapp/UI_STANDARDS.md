# UI Design System: Obsidian & Indigo

## Color Tokens
- Background (Body): #000000 (Pure Black)
- Panels (Sidebars/Cards): #09090b (Dark Charcoal)
- Inputs/Surfaces: #16161c
- Accent: #6366f1 (Electric Indigo)
- Border: rgba(255, 255, 255, 0.05)
- Text Primary: #f8fafc
- Text Secondary (Labels): #94a3b8 (Slate-400)

## Component Rules
- Buttons (Primary): Background Indigo-600, Text White, Rounded-md, Hover: Glow effect.
- Buttons (Secondary/Technical): Outline style, Border Slate-800, Monospace font.
- Typography: Main UI is 'Inter', Technical Data/Logs is 'JetBrains Mono'.
- Spacing: 8px grid (gap-2 = 8px, gap-4 = 16px).
- Border Radius: 8px (rounded-lg) for cards, 4px (rounded-md) for small inputs.

## The Hierarchy Rule
- No floating elements. Everything must live in the 3-Pane Layout.
- Left (210px): Projects. Center: Main Content. Right: Inspector.
